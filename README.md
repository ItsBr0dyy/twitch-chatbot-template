Make sure you have Python 3.8+ installed.
Install the TwitchAPI library with:
```bash
pip install twitchAPI
```

Follow the included setup guide in the Python file to connect your bot to Twitch.

The pre-added commands that are already added include:

- !vanity 
- !chatterino
- !7tv
- etc.

You can add various commands with help from the twitchAPI docs: https://pytwitchapi.dev/en/stable/

If you run into any issues or have questions, feel free to reach out via Discord: @itsbr0dyy —I’m here to help you get your bot running smoothly!
